Truck Work Diary Checker - iPhone PWA prototype

How to use on iPhone:
1. Upload this folder to any HTTPS web hosting service, for example GitHub Pages, Netlify, Vercel, or your own website.
2. Open the website link in Safari on your iPhone.
3. Tap Share.
4. Tap Add to Home Screen.
5. Open it like an app.

What is included:
- Work Diary screen like your screenshots
- 24-hour grid with 15-minute blocks
- BFM Solo and Standard Solo rule checks
- Work, Rest, and Stationary Rest marking
- Live Driving/Rest timer
- Local phone/browser storage
- CSV export

Important:
This is a personal checking tool, not an NHVR-approved Electronic Work Diary.
Keep using your official paper diary or an NHVR-approved EWD.
